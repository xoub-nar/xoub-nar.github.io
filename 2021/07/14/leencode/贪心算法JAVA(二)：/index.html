解题思路：对于中间空位来说，最少需要三个空位才能种一颗花(n-1)/2，对于两端来说，则最少只需要两个空位可满足n/2。

122 买卖股票的最佳时机 II:

给定一个数组 prices ，其中 prices[i] 是一支给定股票第 i 天的价格。设计一个算法来计算你所能获取的最大利润。你可以尽可能地完成更多的交易（多次买卖一支股票）。注意：你不能同时参与多笔交易（你必须在再次购买前出售掉之前的股票）。

示例:

解题思路:

判断第二天的价格是否比前一天的价格高，如果是则继续保留股票，直到股票价格变低则卖出股票。手中无股票时则判断第二天的股票是否比前一天低，低则买入股票。

665 非递减数列：

给你一个长度为 n 的整数数组，请你判断在 最多 改变 1 个元素的情况下，该数组能否变成一个非递减数列。非递减数列： 对于数组中任意的 i (0 <= i <= n-2)，总满足 nums[i] <= nums[i + 1]。

示例：

解题思路：

保证每两个数直接满足后数不小于前数，对于不满足此条件的数(nums[i]<num[i-1]);有两种改变思路，一种是nums[i]=nums[i-1],一种是nums[i-1]=nums[i]。但是要尽量不改变后数，防止因为改变后数而导致后面的数不满足要求。
